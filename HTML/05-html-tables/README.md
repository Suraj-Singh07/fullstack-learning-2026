# Day 3 - HTML Tables

## Objective

Learn how to display structured data using HTML tables.

## Topics Practiced

- Table (`<table>`)
- Table Header (`<thead>`)
- Table Body (`<tbody>`)
- Table Footer (`<tfoot>`)
- Table Rows (`<tr>`)
- Table Headers (`<th>`)
- Table Data (`<td>`)
- Column Groups (`<colgroup>`, `<col>`)
