# Day 9 - CSS Layouts (Flexbox & Grid)

## Objective

Practice modern CSS layout techniques using Flexbox and CSS Grid.

## Topics Practiced

### Display Property

- block
- inline
- inline-block
- flex
- grid

### Flexbox

- display: flex
- flex-direction
- flex-wrap
- justify-content
- align-items

### CSS Grid

- display: grid
- grid-template-columns
- grid-column
- grid-row
- span
- fr unit

## Files

- index.html
- style.css

## Learning Outcome

- Learned the difference between Flexbox and Grid.
- Created multi-column layouts using Grid.
- Used the `fr` unit to divide available space.
- Positioned items with `grid-column` and `grid-row`.
- Expanded grid items using `span`.
